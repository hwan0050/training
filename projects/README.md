# 🚀 실전 프로젝트

> 학습한 내용을 실제 프로젝트에 적용하는 공간

## 📋 프로젝트 목록

### 🛒 E-Commerce Platform (예정)
이커머스 플랫폼을 MSA로 구현하는 프로젝트

**기술 스택:**
- Frontend: Next.js 14, TypeScript, Tailwind CSS
- Backend: Spring Boot, FastAPI
- Database: PostgreSQL, MongoDB, Redis
- Infrastructure: Docker, Kubernetes
- Message Queue: Apache Kafka

**주요 기능:**
- 사용자 인증/인가 (JWT)
- 상품 관리 (CRUD)
- 장바구니 & 위시리스트
- 주문 & 결제
- 리뷰 & 평점
- 실시간 알림

**MSA 서비스 구성:**
- API Gateway
- Auth Service
- User Service
- Product Service
- Cart Service
- Order Service
- Payment Service
- Review Service
- Notification Service

## 🎯 프로젝트 시작하기

### 1. 프로젝트 선택
관심 있는 프로젝트 디렉토리로 이동

```bash
cd projects/e-commerce
# 또는
cd projects/blog-platform
```

### 2. 환경 설정
각 프로젝트의 README.md를 참고하여 환경 설정

### 3. 개발 시작
Git Flow 전략에 따라 브랜치를 생성하고 개발 시작

```bash
git checkout -b feature/user-authentication
```
